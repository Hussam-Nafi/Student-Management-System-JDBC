# 🎓 Student Management System (JDBC)

A Java console application for managing students using JDBC and PostgreSQL.

## 🚀 Features
- View all students
- Add new student
- Update student details
- Delete student
- Search student by ID

## 🛠️ Tech Stack
- Java
- JDBC
- PostgreSQL
- Maven

## 🧠 Concepts Used
- DAO Design Pattern
- PreparedStatement
- ResultSet Mapping
- Try-with-resources

## ⚙️ Setup
1. Create PostgreSQL database
2. Update DBConnection credentials
3. Run Main class

- Console Menu System
